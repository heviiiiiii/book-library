REST endpoints (the two RESTful web-services)

  • GET /api/books — list all books as JSON
  • GET /api/books/{id} — one book as JSON
  • POST /api/books — create a new book (JSON body)
  • PUT /api/books/{id} — replace a book (JSON body)
  • DELETE /api/books/{id} — remove a book

Try curl http://localhost:8081/api/books in a terminal, or open /api/books in a new tab.
